# Muitithread downloader 多线程下载工具

前段时间用了axel感觉很不错，可惜windows下没有，所以自己写了个简化的。

<blockquote>
<pre>
多线程下载器 by yuansu
Usage: mdown.exe [option] URL
  -c string
        要传递的Cookie
  -n int
        同时下载的线程数量[1-50] (default 1)
  -o string
        文件输出位置
  -r string
        引用页
</pre>
</blockquote>

这里提供一个编译好的版本 [win_x64下载](https://github.com/yuansushow/mdown/releases)
